# Timer User Guide

Timer is a command-line chatbot for managing tasks.

## Features

### Add Todo

todo DESCRIPTION

Example:
todo read book

### Add Deadline

deadline DESCRIPTION /by YYYY-MM-DD

Example:
deadline submit report /by 2026-03-20

### Add Event

event DESCRIPTION /from START /to END

### List tasks

list

### Mark a task

mark INDEX

### Unmark a task

unmark INDEX

### Delete a task

delete INDEX

### Find tasks

find KEYWORD

Example:
find book

### Exit

bye
